# 风火山林 v1.56 试玩

桌面端 + 移动端响应式试玩页面。

## 玩法
- 5 角色 + 5 召唤兽 = 10 单位
- 1 回合 = 双方 20 单位按敏捷混排
- 4 职业：风（物理）/ 火（法术）/ 山（治疗）/ 林（封印）

## v1.56 规则（时序系统）
- **时序 1-20**：所有单位按出手顺序行动
- **时序 20+**：复活队列结算
- **昏厥**：跳过该时序的暗扣（不翻开）
- **复活**：从 S.or 移除，追加到复活队列 #1 #2 ...
  - 复活者按**原时序升序排**（8 复活者排 18 复活者前）
  - 复活者**保留原暗扣**（昏厥前已选好）
  - 复活者**清状态**（沉默/混乱/睡眠清除）
- **复活队列全空**才进下一回合

## 操作
1. 点卡 → 弹窗选技能 → 选 1 个目标
2. ✕ 或点空白 → 取消
